# Wasm-Matrix-Hesap-Makinesi

Bu proje, C dilinde yazılmış matematiksel bir hesaplama motorunun WebAssembly (WASM) kullanılarak tarayıcı üzerinde çalıştırılmasıyla oluşturulmuştur.

## Özellikler
- C dili ile yazılmış yüksek performanslı hesaplama motoru.
- İşlem önceliğine duyarlı (parantez destekli).
- Matrix temalı dinamik kullanıcı arayüzü.
- WebAssembly teknolojisi ile tarayıcı içi hızlı hesaplama.

## Teknolojiler
- C
- WebAssembly (Emscripten)
- JavaScript
- HTML/CSS
